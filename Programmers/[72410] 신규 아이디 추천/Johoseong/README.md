# [72410] 신규 아이디 추천
## Algorithm
- 구현, 정규표현식
## Logic
- 주어진 단계별로 구현만 하면 되는 문제.
1. lower() 메소드 사용
2. 정규표현식 ```sub``` 사용 (원하는 문자만 남김)
3. 중복 ```.```을 없애기 위해서, ```.``` 기준으로 spilit() -> 쪼개진 거 탐색하면서 단어면 ```.```으로 이어붙임
```python
tmp = id2.split('.')
id3 = '.'
for word in tmp:
    if not word: continue
    id3 += (word + '.')
```
4. slice로 앞뒤 마침표 제거
나머지 단계는 주어진 그대로 구현하면 됨
