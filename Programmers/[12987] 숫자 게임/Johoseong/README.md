- 단순하게, A와 B를 정렬 후 큰 것부터 차례대로 비교하는 게 최선임.
1. A, B 오름차순 정렬
2. A와 B 스택의 상단 값을 비교해서, B가 이기는 경우면 cnt+1 & 이미 비교한 두 상단값을 제거해줌
3. 만약 B가 지는 경우면, 이 A값은 쓸모 없는 값이니까 그냥 버림. (B의 다른 값들과 비교할 필요 X)
```python
A.sort()
B.sort()

while A:
    if B[-1] > A[-1]:
        answer += 1
        A.pop()
        B.pop()
    else:
        A.pop()
```
