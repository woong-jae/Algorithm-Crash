# [12900] 2 x n 타일링
## Algorithm
- DP
## Logic
- 3이상인 n에 대해, 가로크기 n-1의 경우의 수와 n-2의 경우의 수를 고려해보면 됨
1. n-1의 경우들에서 ```세로로 세워진 타일 하나```만 붙이면 n을 만족하는 경우의 수가 됨
2. n-2의 경우들에서 ```가로로 누워진 타일 두개```만 붙이면 n을 만족하는 경우의 수가 됨
- 즉, ```경우의 수(n) = 경우의 수(n-1) + 경우의 수(n-2)```
```python
    a, b = 1, 2
    for _ in range(3, n + 1):
        c = a + b
        a, b = b, c
```
