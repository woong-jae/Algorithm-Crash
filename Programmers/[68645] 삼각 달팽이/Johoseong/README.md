## Logic
1. 달팽이 모양을 그려보면, 꺾이는 부분(한변의 길이)가 n=6 기준으로 6>5>4>3.. 즉 1씩 줄어듬.
2. 또한 그리는 행위는 맨 처음 ```왼쪽 변 긋기>아래 변 긋기>오른쪽 변 긋기``` 순으로 반복됨.
- 이걸 n*n로 다시 생각해보면, 1의 크기인 각 변마다 2의 순서에 맞춰 칸을 채워주면 됨.
```python
for i in range(n): # 1.
    for _ in range(i, n): # 2.
        if i % 3 == 0:
            r += 1
        elif i % 3 == 1:
            c += 1
        else:
            r -= 1
            c -= 1

        answer[r][c] = cnt
        cnt += 1
```
