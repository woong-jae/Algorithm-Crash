# [2098] 외판원 순회
## Algorithm
- **DP**, **Bitmask**, **TSP**

## Logic
- '외판원 순회' 알고리즘을 사용
  - 도시간의 정보를 저장할 cities, [현재도시 위치][이때까지 방문한 도시들의 비트마스크] 를 저장할 dp배열 생성
- 모든 도시를 방문할 때까지 탐색하면서 최솟값 업데이트
