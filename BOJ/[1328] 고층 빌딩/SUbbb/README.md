# [1328] 고층 빌딩

## Algorithm
- DP

## Logic

```java
dp[n][l][r] = dp[n - 1][l - 1][r] + dp[n - 1][l][r - 1] + dp[n - 1][l][r] * (n - 2)
```

- dp[n][l][r] : n번째 빌딩을 세울 때, 왼쪽에서 보이는 빌딩의 수가 l개, 오른쪽에서 보이는 빌딩의 수가 r개 되도록 건물을 배치하는 경우의 수
- n - 2 : 양 끝을 제외함.
